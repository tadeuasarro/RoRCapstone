## Getting started

### Clone

Clone repo with

```
git clone https://github.com/tadeuasarro/RoRCapstone.git
```

### Setup

Install gems with:

```
bundle install
```

Setup database with:
```
   rails db:create
   rails db:migrate
```

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    bundle exec rspec
```
